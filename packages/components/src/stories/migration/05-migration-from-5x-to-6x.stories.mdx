# Migration from 5.x to 6.x

The following workflow walks through the steps of migrating from version 5.x to 6.x.
Note that the actual steps required for your project may vary, and these steps should be treated as general guidance rather than strict instructions.

## CSS helper classes

- `has-border-radius` use now `has-radius-normal`
- `has-border-blue-dark` use now `has-border-dark`
- `has-border-blue-light` use now `has-border-light`
- `has-no-border` use now `has-border-none`
