# Spacing

In the background [Bulma Spacing Helpers](https://bulma.io/documentation/helpers/spacing-helpers/) is used.

Bulma provides **margin** `m*` and **padding** `p*` helpers for all directions:

- `*t` for **top**
- `*r` for **right**
- `*b` for **bottom**
- `*l` for **left**
- `*x` **horizontally** for both left and right
- `*y` **vertically** for both top and bottom

You need to combine a **margin/padding** prefix with a direction suffix. For example:

- for a `margin-top`, use `mt-*`
- for a `padding-bottom`, use `pb-_`
- for both margin-left and margin-right, use `mx-_`

For **flexbox** there are spacing values also available as `gaps`.

- for a `gap`, use `fg-*`

## Values

Each of these property-direction combinations needs to be appended with one of the following sixteen value suffixes:

| Suffix | Value Small (Mobile) | Value Medium (Tablet) | Value Large (Desktop) |
| ------ | -------------------- | --------------------- | --------------------- |
| `*-0`  | `0`                  | `0`                   | `0`                   |
| `*-1`  | `4px`                | `4px`                 | `4px`                 |
| `*-2`  | `8px`                | `8px`                 | `8px`                 |
| `*-3`  | `12px`               | `12px`                | `12px`                |
| `*-4`  | `16px`               | `16px`                | `16px`                |
| `*-5`  | `20px`               | `20px`                | `24px`                |
| `*-6`  | `24px`               | `24px`                | `32px`                |
| `*-7`  | `32px`               | `40px`                | `48px`                |
| `*-8`  | `48px`               | `56px`                | `64px`                |
