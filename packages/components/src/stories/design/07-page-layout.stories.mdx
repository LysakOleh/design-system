# Page Layout

It is recommended to use following page layout:

```html
<body>
  <bal-app class="has-sticky-footer has-background">
    <main class="container">
      <!-- Page content -->
      <bal-heading>Hello World<bal-heading>
    </main>
    <footer class="footer">
      <!-- Footer content -->
    </footer>
  </bal-app>
</body>
```

- The `.container` is a simple utility class that allows you to center content on larger viewports and to reduce the main space.
  - Use `container is-compact` for sales funnels to reduce the max-width to 768px.
  - [Bulma Documentation to the container](https://bulma.io/documentation/layout/container/)
- `.has-sticky-footer` css-class sticks the footer element to the bottom of the page.
- `.has-background` gives the app a light grey background for better contrast ratios (e.g. in combination with cards)
