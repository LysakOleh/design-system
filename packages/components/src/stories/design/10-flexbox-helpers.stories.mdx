# Flexbox helpers

Helpers for all Flexbox properties

Combined with `is-flex`, all of the Flexbox CSS properties are available as Bulma helpers:

- flex-direction
- flex-wrap
- justify-content
- align-content
- align-items
- align-self
- flex-grow
- flex-shrink

> Check out the [documentation of Bulma](https://bulma.io/documentation/helpers/flexbox-helpers/) for a more details.

## Usage

This section has some examples how to use the provided flexbox css class helpers.

### Centered

To center a child in a parent element use the following classes:

- `is-flex` to enable display flex
- `is-justify-content-center` to set the child horizontally to center
- `is-align-items-center` to set the child vertically to center

<Canvas>
  <bal-doc-app>
    <div className="has-background-blue is-flex is-justify-content-center	is-align-items-center p-3">
      <div className="has-background-danger p-3"></div>
    </div>
  </bal-doc-app>
</Canvas>

```html
<div class="has-background-blue is-flex is-justify-content-center	is-align-items-center p-3">
  <div class="has-background-danger p-3"></div>
</div>
```

### Flexbox Spacing Gaps

On top of the provided flexbox helpers of Bulma we added the flexbox gap classes `fb-*`.

<Canvas>

    <div className="has-background-danger is-flex fg-5">
      <div className="has-background-blue is-flex-grow-1 p-3"></div>
      <div className="has-background-blue is-flex-grow-1 p-3"></div>
      <div className="has-background-blue is-flex-grow-1 p-3"></div>

    <div className="has-background-danger is-flex fg-8 mt-3">
