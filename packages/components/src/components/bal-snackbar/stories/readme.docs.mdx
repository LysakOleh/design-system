# Snackbar

A snackbar is used to inform the user with a simple text message and a action.

<Canvas className="bal-app">
  <bal-snackbar subject="Default" icon="info-circle" action="More">
    Lorem ipsum dolor sit amet, consectetur adipisicing elit.
  </bal-snackbar>
</Canvas>

```html
<bal-snackbar subject="Default" icon="info-circle" action="More"> Lorem ipsum dolor sit amet, consectetur adipisicing elit. </bal-snackbar>
```

## Usage

Snackbar can be created with the balScnackbarController. The default duration is 5000 milliseconds. The create method return the instance of the created snackbar, so there you can access all the props and methods of it.

```typescript
import { balSnackbarController } from '@baloise/design-system-components'

const mySnachbar = balSnackbarController.create({ message: 'Hi I am a default snackbar!', duration: 1000 })
mySnachbar.close()
```
